3).¿Cómo es el manejo de las variables, tipos de datos y expresiones regulares en JS?.

En JavaScript las variables puede definirse de 3 formas:

  • Var
  • Let
  • Const

.Cuando JavaScript fue creado, la unica forma de declarar variables era con el var, el cual tiene, ademas de la opcion de asignarles valor a las variables, el poder de reasignarles un valor. Un ejemplo de esto seria definir "var = "Juan""" y al mostrarlo por pantalla sale el nombre Juan, ahora si volvemos a definir esa variable pero con el nombre de "Abigail", cuando mostremos esa variable por pantalla aparecera el nombre de Abigail.
.La variable Const funciona de casi la misma manera que el var per ocon unos cambios agregados, por decir algunos, el const no puede cambiar de valor, esto quiere decir que si se define con el nombre sofia este no va a poder cambiar de nombre nunca mas. El const tiene alcanze de bloque, esto quiere decir que solo funcionara dentro de una llave de apertura y de salida, si se hace otra llave con el mismo valor el const no funcionara.
.La variable let es todo lo mismo que const per oen este caso las variables si pueden volverse a definir.

4).¿Cómo son todas las formas de declaración de funciones en JS?.

En JavaScript esxisten 3 formas de declaras funciones:

1. Por declaracion:

Es probablemente la forma mas utilizada de declaracion de funciones y la mas facil de recordar. Basicamente consiste en declarar una funcion con un nombre y sus para metros de entrada entre parentesis().

Ejemplo: function add(a, b){ return a+b; } add(5, 6);

2. Por expresion:

Esta forma de declaracion a ganado popularidad con el tiempo y consiste en guardar una funcion en una variable, para luego ejecutar esa variable como una funcion.

Ejemplo: const plus = function (a, b){ return a+b; } plus(5,6);

3. Por constructor de objeto:

La declaracion por constructor de objeto es la menos utilizada y no es recomendada ya su unica funcion es recordar que las funciones son tipos de objetos tambien.

Ejemplo: const greetings = new Function("return 'Hola';"); greetings ();

5).¿Cuáles son los operadores y conversión entre tipos?.

En JavaScript se urilizan 3 tipos de conversiones:

ToString:

Ocurre cuando se muestra algo. El String se realiza con "string(value)".

ToNumber:

Ocurre en operaciones matematicas. El number se realiza con "number(value)".

ToBoolean:

Ocurre en operaciones logicas. El boolean se realiza con "boolean(value)".

Operadores:

Los operadores permiten manipular los valores de las variables, realizar operaciones matematicas con los valores de las variables y comparar diferentes variables.

Tipos de operadores:

Asignacion:

Este operador se utiliza para gusrdar un valor especifico dentro de una variable. Ejemplo: "var num1 = 3;"

Incremento y decremento:

Estos operadores solo son validos para variables numericas y se utilizan para aunmentar o restar en 1 el valor de una variable. El incremento se indica con el prefijo "++" y el decremento con el prefijo "--".
Ejemplo:var num1 = 3;
++num1;
alert(num1);
var num2 = 3;
++num2;
alert(num2);

Logicos:

Los operadores logicos son sumamente necesarios para realizar operaciones complejas, ya que se utilizan para tomar decisiones sobre las instrucciones que deberia ejecutar el programa en funcion a ciertas condiciones.

Negacion:

Se utiliza para obtener el valor contrario al valor de una variable. Ejemplo:var visible = true;
alert(!visible);

AND:

La operacion logica AND obtiene sus resultados combinando dos valores booleanos. El operador se indica con dos && y el resultado solamente sera true si los 2 valores son true. Ejemplo:var valor1 = true;
var valor2 = false;
resultado = valor1 && valor2; //el resultado el "false".
var valor1 = true;
var valor2 = true;
resultado = valor1 && valor2; //el resultado el "true".

OR:

El operador OR funciona de la misma manera que el operador AND pero este tiene la caracteristica de que el resultado sera true si alguno de las 2 valores es true. Tambien este se identifica con 2 palos "||" Ejemplo:var valor1 = true;
var valor2 = false;
resultado = valor1 || valor2; //el resultado el "true".
var valor1 = false;
var valor2 = false;
resultado = valor1 || valor2; //el resultado el "false".

Matematicos:

Permite ralizar operaciones matematicas sobre el valor de las variables numericas. Los operadores que maneja son suma(+), resta(-), multiplicacion(*), division(/), modulo(%). Ejemplo:var num1 = 10;
var num2 = 5;
resultado = num1 + num2;//resultado 15
resultado = num1 - num2;//resultado 5
resultado = num1 * num2;//resultado 50
resultado = num1 / num2;//resultado 2
resultado = num1 % num2;//resultado 0

Relacionales:

Los operadores relacionales estan definidos casi de la misma manera que los operadores matematicos: mayor que (>), menor que (<), mayor o igual (>=), menor o igual(<=), igual que (==), distinto que (!=). Ejemplo:var num1 = 5;
var num2 = 10;
resultado = num1 > num2;//resultado "false"
resultado = num1 < num2;//resultado "true"
resultado = num1 >= num2;//resultado "true"
resultado = num1 <= num2;//resultado "true"
resultado = num1 == num2;//resultado "true"
resultado = num1 != num2;//resultado "false"

6).¿Cuáles son las estructura de control de JS?.

Estructura de control y sus tipos:

Una estructura de control de datos es una herramienta que evita que un programa haga siempre lo mismo, dandole la posibilidad de hacer cambios segun las condiciones que se den.

if/else:

La estructura if/else permite ejecutar un bloque de codigo si la condicion es verdadera y otro bloque de codigo si la condicion es falsa.
Ejemplo if:
if(condicion){
//instrucciones si la condicion es verdadera
}
Ejemplo else:
if(condicion){
//instrucciones si la condicion es verdadera
}else{
//instrucciones si la condicion es falsa
}

Ejemplo multipleelse:
if(condicion){
//instrucciones si la condicion 1 es verdadera
}elseif(condicion2{
//instrucciones si la condicion 1 es falsa y la condicion 2 es verdadera
}else{
//instrucciones si la condicion 1 y 2 son falsas
}

Switch

Consiste en definirle un valor al switch y ver si ese valor coincide con alguno de los casos que se pueden definir dentro de este, si un caso resulta ser verdadero, este ejecutara el bloque de codigo que tenga definido.
Ejemplo:
switch(valor){
case valor_1:
// Instrucciones para el valor_1
break;
case valor_2:
// Instrucciones para el valor_2
break;
case valor_n:
// Instrucciones para el valor_n
break;
default:
// Instrucciones en caso de que no se cumpla ninguna condición
}

For:

Ejecuta codigo un numero determinado de veces.
Ejemplo:
for(inicialización; condición; actualización){
// Código que se repite
}

While:

Ejecuta un bloque de codigo siempre y cuando se cumpla la condicion que se indica entre parentesis.
Ejemplo:
while(condición){
//Código a ejecutar
}
Ejemplo do/while:
do{
// Código a ejecutar
}while(condición);

Break y continue:

Break significa salir de un bucle y continue significa saltar hasta la siguiente repeticion de un bucle.
Ejemplo break:var i = 0;
for(i = 0; i < 10; i++){
if (i == 3){ // Cuando llega al 3 se ejecuta el break y sale del bucle for
break;
}
Ejemplo continue:
var j = 0;
for(j = 0; j < 10; j++){
if (j == 3 || j == 5){
continue; // Saltar iteración si j vale 3 o 5 (no ejecutará el alert)
}

8).¿Qué función tiene en JS el Object Map?.

Object map:

El metodo array.map te permite iterar(realizar una accion varias veces) sobre un arreglo y modificar sus elementos con la funcion callback. El callback se ejecuta en cada elemento del arreglo.
Ejemplo de un for:
let arr = [3, 4, 5, 6];
for (let i = 0; i < arr.length; i++){
arr[i] = arr[i] * 3;
} console.log(arr); // [9, 12, 15, 18]
esto seria una multiplicacion usando for, pero ahora veremos una con array.map y veremos que se obtendra el mismo resultado: let arr = [3, 4, 5, 6];
let modifiedArr = arr.map(function(element){
return element *3;
});
console.log(modifiedArr); // [9, 12, 15, 18]
