2).¿Cómo se integra JavaScript a HTML?.

Para poder integrar JS en nuestro archivo HTML podemos hacerlo con la etiqueta "script" la cual te permite insertar dentro de ella codigo JS. El "script" se puede usar dentro del head (para que el script permanezca alejado del contenido del documento HTML) o dentro del body (para mejorar la velocidad de carga y que el contenido del sitio webcargue mas rapido).

3).¿Cómo es el manejo de las variables, tipos de datos y expresiones regulares en JS?.

En JavaScript las variables puede definirse de 3 formas:

  • Var
  • Let
  • Const

.Cuando JavaScript fue creado, la unica forma de declarar variables era con el var, el cual tiene, ademas de la opcion de asignarles valor a las variables, el poder de reasignarles un valor. Un ejemplo de esto seria definir "var = "Juan""" y al mostrarlo por pantalla sale el nombre Juan, ahora si volvemos a definir esa variable pero con el nombre de "Abigail", cuando mostremos esa variable por pantalla aparecera el nombre de Abigail.
.La variable Const funciona de casi la misma manera que el var per ocon unos cambios agregados, por decir algunos, el const no puede cambiar de valor, esto quiere decir que si se define con el nombre sofia este no va a poder cambiar de nombre nunca mas. El const tiene alcanze de bloque, esto quiere decir que solo funcionara dentro de una llave de apertura y de salida, si se hace otra llave con el mismo valor el const no funcionara.
.La variable let es todo lo mismo que const per oen este caso las variables si pueden volverse a definir.

JavaScript acepta unos 8 tipos de datos, los cuales van a definir en la siguiente tabla:

Number Representa tanto numeros enteros como numeros flotantes
Bigint Permite aumentar la cantidad de numeros que se pueden ingresar en el numbre (253-1) y -(253-1)
String Define una cadena de carcteres
Boleano dato de tipo logico que acepta 2 valores posibles: true o false
Null No pertenece a ningun dato definido hasta ahora
Undefined Funciona igual que null
Object Sirve para coleccionar colecciones de datos y otros elementos mas complejos, es decir, puede tener muchos datos dentro de uno solo
Symbol Sirve para crear identificadores unicos para los objetos

Expresion regular:

Una expresion regular es un patron que define un conjunto de cadenas sin enumerar todos sus elementos, estas expresiones regulares estan formadas de caracteres y metacaracteres que tienen su funcion definida. En JavaScript existen 2 formas de definir una expresion regular. 1 es mediante una cadena literal y el otro mediante un constructor de objeto.
